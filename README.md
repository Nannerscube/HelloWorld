# Hello world! Chess Bot

Made in Python and C++ for Arduino Uno Q

Outputs:
- Recognize chess board
- Waits player to play
- Decides best move
- Plays best move by himself

## Requirements
- [Arduino UNO Q 2GB/4gb](https://www.arduino.cc/product-uno-q)
- Robot Arm
- Webcam 1080p wide angle
- [Stockfish](https://stockfishchess.org/download/)

- [Visual Studio Code](https://code.visualstudio.com/download)

## How It Works
The hello world chess robot is a robot arm system that plays chess. It uses image recognision to look at chess board states and decides on its own what move to make.
